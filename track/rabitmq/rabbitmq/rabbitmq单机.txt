1、安装 erlang
	yum 安装
		yum install -y erlang
	rpm 安装
		下载 
			erlang-19.3.6.12-1.el7.centos.x86_64.rpm
		安装：
			rpm -ivh erlang-19.3.6.12-1.el7.centos.x86_64.rpm
2、安装 rabbitmq 
	yum安装：
		yum install -y rabbitmq-server
	rpm 安装
		下载 
			rabbitmq-server-3.7.7-1.el7.noarch.rpm
		安装依赖：
			yum -y install socat
		安装rabbitmq:
			rpm -ivh rabbitmq-server-3.7.7-1.el7.noarch.rpm
		地址：
			github地址：	https://github.com/rabbitmq/rabbitmq-server/releases
			官网地址：		http://www.rabbitmq.com/releases/rabbitmq-server/

3、启动rabbitmq
	命令：
		systemctl start rabbitmq-server
4、启用页面插件：
	命令：	rabbitmq-plugins enable rabbitmq_management
5、添加用户：
	# 添加 账号：admin 密码：admin 的用户，并赋予 administrator 的权限
	rabbitmqctl add_user admin admin
	rabbitmqctl set_user_tags admin administrator
