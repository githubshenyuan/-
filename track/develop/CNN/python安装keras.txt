1、window上安装 python keras
	环境：python3.6.6 （3.7.0 版本不支持）
	命令：
		# GPU 版本
		pip3 install --upgrade tensorflow-gpu
		# CPU 版本
		pip3 install --upgrade tensorflow
		# Keras 安装
		pip3 install keras -U --pre
	国内镜像：
		# GPU 版本
		pip3 install --upgrade tensorflow-gpu -i http://pypi.douban.com/simple
		# CPU 版本
		pip3 install --upgrade tensorflow -i http://pypi.douban.com/simple
		# Keras 安装
		pip3 install keras -U --pre -i http://pypi.douban.com/simple
